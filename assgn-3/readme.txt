2. What is DOCTYPE in Html ?

    It is a declaration used to tell the browser the version of the markup language being used.


3. What is Void Element and Example for Void Elements ?

    Void elements do not have any content or child. They do not have the closing tag.
    Eg: <br>,<hr>


4. What is Different Element and Attributes ?

    Element is a basic block that tells the broser what type of content to display while attributes are the additional properties to modify an element.
    Eg: <img src="">
        where <img> is an element and 'src' is an attribute.


5. What is Html Entries and Why is it need in HTML ?

    HTML Entries are special codes used to display reserved characters, symbols, and invisible spaces. Some characters cannot be typed directly as it might be misinterpreted by the browser as the code or in a different meaning.
    It is maked using '&' in the beginning and ';' in the end. when using the unicode entity numbers to write symbols '#' is used in the  numerical code format and '#x' is used in the hexadecimal format.
    Eg: For copyright symbol(©) 
        '&copy;' in entity name 
        '&#169;' in decimal values  and 
        '&#xA9;' in the hexadecimal values any of them can be used tto bring the symbol in the text content in our page.


6. What are meta tag and why is it used?

    Metadata refers to information that describes an HTML document—such as its character encoding, page description, keywords, and author—without being visible on the webpage itself.
    It is used for Search Engine Optimization, Browser Rendering.
