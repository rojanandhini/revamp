Food Store Web Application (with Reservation System)

  • Developed a responsive food store web application using HTML, CSS, and JavaScript.
  • Implemented dynamic product catalog and shopping cart using JavaScript DOM manipulation and event handling.
  • Designed a reservation system allowing users to:
    • Select date and time slots
    • Add number of adult and child guests
    • Automatically calculate total cost based on guest count
  • Integrated Firebase Realtime Database to store and retrieve reservation data in real time.
  • Implemented logic to disable (grey out) previously booked time slots to prevent double bookings.
  • Generated checkout bill summary with guest-wise cost breakdown.
  • Applied real-time validation and conditional rendering without page reload.
  • Ensured responsive design and smooth user experience across devices.
